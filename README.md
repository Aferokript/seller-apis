# Программа для продавцов на маркетплейсах "Ozon" и "Яндекс Лавка"

## seller.py

Программа для работы с маркетплейсом "Ozon". Обладает следующим функционалом:

- Получить список товаров. 
- Получить арктикул товаров
- Может обновить цену товара
- Может обновить цену остатков товара
- Позволяет задавать цену на ваш товар
- Убрать из цены лишние символы
 
## market.py

Программа для работы с маркетплейсом "Яндекс Маркет". Обладает следующим функционалом:

- Получить список товаров. 
- Получить арктикул товаров
- Может обновить цену товара
- Может обновить цену остатков товара
- Позволяет задавать цену на ваш товар
- Убрать из цены лишние символы
